# Interview Repo

면접 준비 과정에서 발생하는 질문과 답변을 체계적으로 관리하기 위한  
**면접 질문 관리 웹 플랫폼**입니다.

직무별·기술별 질문을 구조화하고,  
개인별 답변과 학습 이력을 함께 관리하는 것을 목표로 설계했습니다.

---

## 프로젝트 목적

- 흩어져 있는 면접 질문 및 답변을 구조화된 데이터로 관리
- 직무·기술 기준의 질문 분류 및 검색 기능 제공
- 면접 준비 과정에서의 반복 학습과 복습 효율 개선
- 프론트엔드–백엔드 협업 구조 경험 및 실서비스 확장 고려

---

## 주요 기능

### 면접 질문 관리
- 직무별 / 기술별 면접 질문 CRUD
- 질문 단위로 개인 답변 저장 가능
- 실사용을 고려한 데이터 모델 설계

### 검색 및 분류
- 키워드 기반 검색
- 태그 기반 질문 필터링
- 북마크 기능을 통한 중요 질문 관리

### 사용자 기능
- 회원가입 / 로그인
- 사용자별 질문 및 답변 데이터 분리
- 반응형 UI 지원 (모바일 / 데스크톱)
